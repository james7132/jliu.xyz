# jliu.xyz

My personal website. Built with [zola](https://www.getzola.org/),
[SponsorKit](https://github.com/antfu/sponsorkit), and GitHub Pages.

## Building
The entire site can be built with `zola build`. Editing can be done live with
`zola serve`.
